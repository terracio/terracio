## Open Source

### policy-eval-harness
**A reference methodology and evaluation harness for LLM agents in sequential environments.**

A deterministic replay + scorecard + promotion-gate framework that replaces subjective reviews with explicit, reproducible, business-aligned evaluation.

- Deterministic replay of policies on identical evaluation cases  
- Scorecards with utility, behavior, and reliability metrics  
- Explicit promotion gates (`PROMOTE` / `REJECT`) based on quantitative deltas  
- Designed for sequential, tool-using agents (fraud, claims, underwriting, operations, etc.)  
- Full audit trail and CI/CD-ready artifacts  

[View repo →](https://github.com/terracio/policy-eval-harness)

This repo reflects how I approach enterprise agent reliability: explicit evaluation, reproducible comparison, and promotion decisions grounded in measurable outcomes.
